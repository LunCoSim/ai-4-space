Ethereum Virtual Machine (EVM) compatible chains use the same address format (`0x...`).

## Wallets
- **[Rabby](https://rabby.io)**: 
  - *Best Feature*: Pre-transaction security checks.
  - *Management*: Handles dozens of chains automatically.
- **[MetaMask](https://metamask.io)**: 
  - *Best Feature*: Found everywhere.
  - *Management*: Requires manual network adding sometimes (use [Chainlist](https://chainlist.org)).

## Core L2 Networks
Layer 2s (L2s) are built on top of Ethereum to make it cheap.
- **[Base](https://base.org)**: Built by Coinbase. Standard for new projects.
- **[Polygon](https://polygon.technology)**: Fast, low fees, mature ecosystem.
- **[Arbitrum](https://arbitrum.io)**: Highly efficient, popular for DeFi.

## Gas Fees
- You pay fees in the **Native Token** of that chain.
- Ethereum -> **ETH**
- Base -> **ETH**
- Polygon -> **POL (formerly MATIC)**
- Arbitrum -> **ETH**
